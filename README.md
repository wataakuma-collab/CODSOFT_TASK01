# My Portfolio

This is my personal portfolio site. Plain HTML and CSS— no frameworks, no build steps, nothing fancy under the hood. Just wanted something that loads fast and works anywhere.

I'm a CSE final year student, currently interning as a web dev. Learning React next, but for now this is pure vanilla frontend.

## What's in it

- About me
- Skills
- Projects
- Resume (downloadable)
- Contact info

## Files

- `index.html` — the whole site
- `profile.jpg` — my photo
- `resume.pdf` — my resume

## Running it

Just open `index.html` in your browser. That's it, no setup needed.

```bash
git clone https://github.com/wataakuma-collab/portfolio.git
```
